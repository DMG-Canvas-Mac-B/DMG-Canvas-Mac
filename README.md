## 📋 Overview

<a href="#dMGCanvas">DMG Canvas</a> serves Mac developers who distribute their applications as DMG files and want professional-looking installers, build engineers who need automated DMG creation in release pipelines, and any developer who needs to meet Apple's current distribution requirements for code-signed, notarized DMG files.

The <a href="#dMGCanvas">indie Mac developer use case</a> creates polished DMG installers that match the quality of the application itself — a professionally designed installer window with branded background, correctly positioned icons, and embedded license text communicates attention to quality that a plain unstyled DMG does not. The <a href="#dMGCanvas">automated build pipeline use case</a> integrates DMG Canvas into CI/CD workflows — the command-line interface builds DMGs from saved project files automatically on each release, producing consistent signed and notarized DMGs without manual intervention.

The <a href="#dMGCanvas">commercial software distribution use case</a> handles the complete distribution package — signed application, notarized DMG, embedded EULA, compressed archive — meeting all requirements for distributing a commercial Mac application outside the Mac App Store. <a href="#dMGCanvas">Notarization requirement use case</a> — since Apple's Gatekeeper now rejects unsigned and unnotarized applications on most Macs, DMG Canvas's integrated signing and notarization support makes meeting these requirements straightforward rather than requiring manual multi-step workflows.

## 🤔 FAQ

<details>
<summary>What is notarization and why is it required?</summary>

Apple's notarization service scans applications for malicious content. Notarized applications and DMGs pass Gatekeeper on recipient Macs without warning dialogs — required for practical distribution outside the Mac App Store.

</details>

<details>
<summary>Can I save DMG configurations for reuse?</summary>

Yes. Project files save the complete DMG configuration — window size, icon positions, background, license, signing settings — for reproducible builds.

</details>

<details>
<summary>Does DMG Canvas support Retina displays?</summary>

Yes. 2x resolution background images render sharply on Retina and high-DPI displays.

</details>

<details>
<summary>What is the internet-enable option?</summary>

Internet-enable configures a DMG for automatic expansion when downloaded from the web — standard practice for web-distributed Mac DMG files.

</details>

<details>
<summary>Can multiple developers share DMG Canvas project files?</summary>

Yes. Project files save all configuration settings and work on any Mac with DMG Canvas installed.

</details>
